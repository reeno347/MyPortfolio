Water Monitoring System Based on Internet of Things (IoT) Using Telegram Application

Along with the development of the times as it is today, a lot of technology has emerged to make it easier to help human work quickly. One of them is the Internet of Things (IoT), this technology is very likely to be a hope in the future. Internet of Things (IoT) is a concept that aims to expand the benefits of continuous internet connectivity. Simply put, with IoT, objects in the real world can communicate with each other using the help and internet network. The access capabilities of IoT can be unlimited thanks to IoT devices that are always connected to the internet, so they can be accessed and used anytime and anywhere. Based on these problems, the author is encouraged to raise the title “Monitoring System for Water Worthy of Consumption Based on the Internet of Things (IoT) Using Telegram Application” to determine the quality of the water.

The purpose of making tools Designing an IoT-based consumable water monitoring system using ESP32. Creating an IoT-based consumable water monitoring system to make it easier for the community and drinking water company (PDAM) officers when controlling water sources. Testing the existing application on a smartphone in telegram so that it can receive and store data from the microcontroller system.
